# @marcobiedermann/tsconfig

Shared [`tsconfig`](https://www.typescriptlang.org/docs/handbook/tsconfig-json.html) for my projects.

## Install

```sh
npm install --save-dev @marcobiedermann/tsconfig
```

## Usage

`tsconfig.json`

```json
{
  "extends": "@marcobiedermann/tsconfig",
  "compilerOptions": {
    "outDir": "./dist"
  }
}
```
